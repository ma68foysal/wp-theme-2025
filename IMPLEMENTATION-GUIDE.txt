================================================================================
TROUBLESHOOTING:
================================================================================

Search not working?
  - Check AJAX nonce in browser console
  - Verify bantuAjax object exists
  - Check wp_ajax hook registration

Videos not displaying?
  - Verify video post type is registered
  - Check featured images are set
  - Confirm Bunny.net GUID stored in meta

Dashboard not accessible?
  - Verify page is created with dashboard slug
  - Check user is logged in
  - Verify shortcode is placed on page

Payment issues?
  - Check Stripe keys are configured
  - Verify webhook URL is correct
  - Check membership expiry format (timestamp)

Mobile menu not working?
  - Verify navigation menu is assigned
  - Check media query breakpoints
  - Test on actual device (not just browser dev tools)
