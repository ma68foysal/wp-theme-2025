INSTALLATION STEPS:
================================================================================

1. COPY THEME FILES
   └─ All files already in /vercel/share/v0-project/

2. CREATE PAGES IN WORDPRESS ADMIN:
   
   a) Homepage:
      - Go to Settings > Reading
      - Set "Front Page" to "Your Front Page"
   
   b) Dashboard Page:
      - Create new page titled "Dashboard"
      - Slug: dashboard
      - Content: Leave blank (uses template)
      - Publish
   
   c) Browse/Videos Page:
      - Create new page titled "Browse"
      - Slug: videos
      - Publish

3. CREATE NAVIGATION MENU:
   - Go to Appearance > Menus
   - Create new menu "Main Menu"
   - Add items: Home, Browse, Dashboard (if logged in)
   - Set location: Primary Menu

4. ADD SAMPLE VIDEOS:
   - Go to Videos > Add New
   - Title: "Your Video Title"
   - Set featured image (thumbnail)
   - Select category: Action, Drama, Comedy, etc.
   - Add meta: video_duration = "45:00" (format)
   - Publish (repeat 5-10 times)

5. CONFIGURE BUNNY.NET (in Admin):
   - Go to Dashboard > BANTU Plus > Settings
   - Enter Bunny.net API Key
   - Enter Library ID
   - Save

6. CONFIGURE STRIPE (in Admin):
   - Go to Dashboard > BANTU Plus > Settings
   - Enter Stripe Public Key
   - Enter Stripe Secret Key
   - Enter Webhook Secret
   - Save

DONE! Your site is ready.

API ENDPOINTS (if using mobile app):
================================================================================

GET /wp-json/bantu/v1/videos
- Get all videos with pagination
- Parameters: page, per_page, category, sort

GET /wp-json/bantu/v1/videos/{id}
- Get single video details

POST /wp-json/bantu/v1/auth/login
- User login
- Body: username, password

GET /wp-json/bantu/v1/user/history
- Get watch history (requires auth token)

POST /wp-json/bantu/v1/user/progress
- Save video progress (requires auth token)


TROUBLESHOOTING:
================================================================================

Issue: Search not working
Fix:
1. Clear browser cache
2. Check WordPress AJAX is enabled
3. Verify nonce in console (F12 > Console)
4. Check bantuAjax object exists

Issue: Videos not showing on homepage
Fix:
1. Create Videos with featured images
2. Add categories to videos
3. Check video post type is enabled
4. Clear WordPress cache

Issue: Dashboard not accessible
Fix:
1. Create "Dashboard" page with slug /dashboard
2. Check user is logged in
3. Verify page is published

Issue: Mobile not responsive
Fix:
1. Clear browser cache
2. Hard refresh (Ctrl+F5)
3. Check viewport meta tag in header
4. Test on actual device (not just dev tools)

Issue: User menu not showing
Fix:
1. Check header.php is being used
2. Verify is_user_logged_in() function works
3. Check avatar URL loading
4. Clear browser cache
